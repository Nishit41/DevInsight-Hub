# Express
Express is a web application framework for Node.js. It provides utility functions to easily define routes for REST APIs and serve HTTP servers.

# body-parser module
This module detects if a client sent a JSON request (by looking at the Content-Type header) and then automatically parses it for us so that we can access the object in req.body.
